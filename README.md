Full Stack Budget & Experiment Integration Website
📌 Project Overview
This project is a full-stack integrated web application that combines multiple experiments into a single unified website with
seamless navigation and functionality.
---------------------------------------------------------------------------------------------------------------------------------
The system includes:
Budget tracking module
Transaction and history management
Analytics dashboard
Feedback system (Java Servlets)
JSP-based utilities
XML reports
Node.js and React integrations
All modules are connected through a global sidebar navigation, making the application easy to use and test.
